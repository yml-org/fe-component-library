# List Component

The List component is a Lit component that renders a list component that is framework independent and utilizes Tailwind CSS internally.

# Examples

Ordered List default

export const orderedListTemplate = ({ listType, listItems }) =>
  html`<list-component .listType=${listType} .listItems=${listItems} />`;

<Canvas>
  <Story name="ordered list" args={orderedListVariant}>
    {orderedListTemplate.bind({})}
  </Story>
</Canvas>
