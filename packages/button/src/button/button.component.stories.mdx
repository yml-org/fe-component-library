# Button Component

The Button component is a Lit component that renders a button element that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name  | Type        | Description                                                                          |
| ---------- | ----------- | ------------------------------------------------------------------------------------ |
| text       | string      | The text for the button element                                                      |
| linkTitle  | VariantType | The button variant                                                                   |
| color      | colorType   | The button background color                                                          |
| slotName   | string      | The name for the slot that can be used to pass children                              |
| isDisabled | boolean     | Flag to determine whether the button needs to be disabled or not                     |
| fullWidth  | boolean     | Flag to determine whether the button needs to occupy the full width of the container |
| isRounded  | boolean     | Flag to determine whether the button will have rounded corners                       |
| partName   | string      | Custom part name provided by user for styling                                        |

The button has the following variants:

1. contained
2. outlined
3. link

The different color options for the button component are as follows:

1. primary
2. secondary
3. success
4. danger
5. warning
6. light
7. dark
