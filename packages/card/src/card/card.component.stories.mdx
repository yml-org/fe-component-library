# Card Component

The card component is a Lit component that renders a skeleton card that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name         | Type              | Description                                                |
| ----------------- | ----------------- | ---------------------------------------------------------- |
| hasRoundedBorder  | boolean           | Flag to determine whether or not to have rounded borders   |
| cardWidth         | number            | The width of the card component                            |
| cardHeight        | number            | The height of the card component                           |
| customSlotName    | string            | The name of the child slot                                 |
| showCustomSlot    | boolean           | Flag to determine whether or not to render the custom slot |
| cardDimensionUnit | CardDimensionUnit | The unit for the card width and height                     |
| isHorizontal      | boolean           | Flag to determine the content direction                    |

Following are the values for the CardDimensionUnit:

1. px
2. %
3. em
4. rem
5. pt
6. cm
7. in
8. mm
9. pc
10. ex
11. ch
12. vh
13. vw

The default slots provided in the component are as follows:

1. webcl-card-header
2. webcl-card-content
3. webcl-card-footer
