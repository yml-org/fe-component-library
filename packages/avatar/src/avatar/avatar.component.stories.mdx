# Avatar Component

The Avatar component is a Lit component that renders an avatar component that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name       | Type     | Description                                                              |
| --------------- | -------- | ------------------------------------------------------------------------ |
| isRounded       | boolean  | Flag to determine whether or not to have rounded borders                 |
| width           | number   | The width of the avatar component                                        |
| height          | number   | The height of the avatar component                                       |
| usePercentage   | boolean  | Flag to determine whether or not the dimensions should be in percentages |
| hasBorder       | boolean  | Flag to determine whether the border is shown or not                     |
| borderColor     | string   | The border color                                                         |
| slotName        | string   | The name of the child slot                                               |
| onAvatarClick   | Function | The click function that gets triggered on avatar component click         |
| hasShadow       | boolean  | Flag to determine whether the avatar component has box shadow or not     |
| showDefaultIcon | boolean  | Flag to determine whether tho show default avatar icon or not            |
| defaultImage    | string   | The default image to be displayed                                        |
| defaultBgColor  | string   | The default background color provided by the user                        |
