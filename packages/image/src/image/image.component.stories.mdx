# Image Component

Basic Image component with `src` prop

Here we include `src` prop to srcset attribute
with a reference to a high-resolution version
f the image. this will be loaded instead of the src
image on high-resolution devices.

export const defaultTemplate = ({ src }) =>
  html`<image-component .src=${src}></image-component>`;

<Canvas>
  <Story
    name="default"
    args={{
      src: 'https://i.ibb.co/yBV3M1C/planet.jpg',
    }}
  >
    {defaultTemplate.bind({})}
  </Story>
</Canvas>

export const withAltTextTemplate = ({ src, altText }) =>

Image component with `placeholderImg` prop

In this example we include `placeholderImg` prop to
src attribute. Displayed, in case the 'srcset' is
failed to load the image or before image is loaded

export const withPlaceholderImgTemplate = ({ src, altText, placeholderImg }) =>
  html`<image-component
    .src=${src}
    .altText=${altText}
    .placeholderImg=${placeholderImg}
    ><image-component></image-component
  ></image-component>`;

<Canvas>
  <Story
    name="PlaceHolder Image"
    args={{
      src: 'https://i.ibb.co/yBV3M1C/planet.jpg',
      altText: 'background image',
      placeholderImg: 'https://i.ibb.co/yBV3M1C/planet-mini.jpg',
    }}
  >
    {withPlaceholderImgTemplate.bind({})}
  </Story>
</Canvas>

Image component with `width`, `height`

  html`<image-component
    .src=${src}
