# ProgressBar Component

The progress bar component is a Lit component that renders a progress bar with dynamic percent value that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name                | Type   | Description                          |
| ------------------------ | ------ | ------------------------------------ |
| percent                  | number | Progress percent of the progress bar |
| max                      | number | Maximum percent of the progress bar  |
| containerPartAttribute   | string | The part name for container tag      |
| progressBarPartAttribute | string | The part name for progress bar       |

# Examples

Basic progress bar component with percent

export const TemplateWithProgressPercent = ({ percent }) =>
  html`
    <div class="progress-bar-storybook-container">
      <ymlwebcl-progressbar .percent=${percent}></ymlwebcl-progressbar>
    </div>
  `;

<Canvas columns={1}>
