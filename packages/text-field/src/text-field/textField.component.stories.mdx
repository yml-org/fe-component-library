# TextField Component

The text field component is a Lit component that renders a skeleton text field that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name              | Type         | Description                                                         |
| ---------------------- | ------------ | ------------------------------------------------------------------- | --- |
| label                  | string       | Text field label                                                    |
| textFieldId            | string       | The id for the text field                                           |
| value                  | string       | The value of the text field                                         |
| placeholder            | string       | The placeholder for the text field                                  |
| hintText               | string       | The hint text for the text field                                    |
| autocomplete           | AutoComplete | The autocomplete attribute for the text field                       |
| autofocus              | boolean      | Flag to determine whether the text field will be focussed initially |
| disabled               | boolean      | Flag to determine whether the text field will be disabled           |
| readonly               | boolean      | Flag to determine whether the text field will be readonly           |
| required               | boolean      | Flag to determine whether the text field will be required field     |
| labelPartAttribute     | string       | The part name for label tag                                         |
| iconPartAttribute      | string       | The part name for container tag                                     |
| hintTextPartAttribute  | string       | The part name for hint text tag                                     |     |
| textFieldPartAttribute | string       | The part name for text field tag                                    |
| minLength              | number       | The min length for the text field content                           |
| maxLength              | number       | The max length for the text field content                           |
| clearField             | boolean      | Flag to clear text field                                            |
| error                  | boolean      | Flag to indicate error in input field                               |
| changeHandler          | string       | The name of the custom change handler event                         |
| clickIconHandler       | string       | The name of the custom click handler event                          |

The default slots provided in the component are as follows:

1. textfield-icon
