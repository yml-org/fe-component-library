# TextArea Component

The text area component is a Lit component that renders a textarea that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name              | Type         | Description                                                        |
| ---------------------- | ------------ | ------------------------------------------------------------------ |
| label                  | string       | Text area label                                                    |
| textAreaId             | string       | The id for the text area                                           |
| textAreaName           | string       | The name of the text area                                          |
| textAreaRows           | number       | The number of rows for the textarea                                |
| textAreaCols           | number       | The number of columns for the textarea                             |
| textAreaAutoComplete   | AutoComplete | The autocomplete attribute for the textarea                        |
| isAutoFocus            | boolean      | Flag to determine whether the text area will be focussed initially |
| isDisabled             | boolean      | Flag to determine whether the text area will be disabled           |
| isReadOnly             | boolean      | Flag to determine whether the text area will be readonly           |
| isRequired             | boolean      | Flag to determine whether the text area will be required field     |
| labelPartAttribute     | string       | The part name for label tag                                        |
| containerPartAttribute | string       | The part name for container tag                                    |
| textAreaPartAttribute  | string       | The part name for textarea tag                                     |
| formId                 | string       | The id for the form tag the textarea tag is a part of              |
| textAreaMaxLength      | number       | The max length for the text area content                           |
| textAreaMinLength      | number       | The min length for the text area content                           |
| textAreaPlaceHolder    | string       | The placeholder text for the text area                             |
| changeHandlerName      | string       | The name of the custom change handler event                        |
| canResize              | boolean      | Flag to determine whether or not the textarea can be resized       |

The AutoComplete type has the following values:

1. off
2. on
