# Chip Component

The Chip component is a Lit component that renders a chip component that is framework independent and utilizes Tailwind CSS internally.

# Props

| Prop Name              | Type   | Description                                                                                   |
| ---------------------- | ------ | --------------------------------------------------------------------------------------------- |
| text                   | string | text within the chip component                                                                |
| chipPartAttribute      | string | The part name for chip component                                                              |
| rightSlotName          | string | Name given to slot that renders an HTMLElement on the right of the text within chip component |
| leftSlotName           | string | Name given to slot that renders an HTMLElement on the left of the text within chip component  |
